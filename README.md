# Autor: Bruno

📊 Sistema de Análise de Vendas

Projeto desenvolvido em Python com foco em prática de lógica de programação e análise de dados.

🔹 Funcionalidades

- Tratamento de entrada de dados (remoção de R$, espaços e formatação)
- Cálculo de lucro da loja
- Soma de vendas por período
- Verificação de meta de faturamento

🔹 Tecnologias utilizadas
 - Python

🔹 Exemplo de uso
Digite o valor: R$ 3000
Lucro da loja: R$ 2200,00
Deu lucro

Vendas segunda: 1000
Vendas terça: 2000
Vendas quarta: 1500

Total de Vendas: R$ 4500,00
Meta não atingida.

🔹 Objetivo
Praticar conceitos fundamentais de programação e simular uma análise simples de vendas, similar a tarefas iniciais de um Analista de Dados.
